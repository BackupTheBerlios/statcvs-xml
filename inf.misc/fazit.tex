\section{Zusammenfassung}

\frame{
  \frametitle{Zusammenfassung}

  Maven ist ein tolles Tool
  
  \begin{itemize}
  \item Sehr einfache Erstellung von Projekt Seiten
  \item Sehr einfache Erhebung von Projekt und Prozess Metriken
  \item Sehr einfaches Build und Release
  \item Sehr einfaches Deployment 
  \item Automatische Auflösung von Abhängigkeiten
  \end{itemize}  
    
  Manches ist noch nicht ganz ausgereift

  \begin{itemize}
  \item Lange Startzeiten (auf marvin ca. 16s)
  \item Viele Reports sind Java spezifisch
  \item Anpassung der Webseiten schwierig
  \item Keine Unterstützung für I18n
  \item Generierte Webseiten enthalten viele Leerzeilen
  \end{itemize}
}
